对 **VTube Studio** 支持哪些 **VTuber 模型类型** 有疑问？或者你正在**寻找可以使用的模型**？此页面将帮助你入门。

# 我可以在 VTube Studio 中使用哪些模型？

VTube Studio **仅支持Live2D模型**.

不支持 VRoid/VRM 等其他格式。 VTube Studio **不支持任何 3D 模型**。对于 3D 模型，你可以查看 VSeeFace：https://www.vseeface.icu/

# 在哪里可以获得 Live2D 模型？或者我怎样才能制作它们？

一般来说，Live2D 模型不是免费的。你必须委托一位画师为你的模型绘制**带有拆分的**立绘，然后你必须委托 Live2D 建模师使用该立绘创建模型。这个过程通常需要几周到几个月的时间，完整的模型成本可能在几百到几千元之间，因为它们是由画师根据你的要求从头开始制作的。

VTube Studio **没有**内置建模器。

如果你想委托制作模型，Live2D Discord 是寻找画师和建模师的好地方: https://discord.gg/Live2D

或bilibili工坊、米画师、画加等约稿平台。

如果你已经有了模型立绘想要自己制作模型，你可以从这里下载Live2D Cubism: https://www.live2d.com/zh-CHS/

YouTube、BiliBili上有很多教程可帮助你开始使用 Live2D Cubism。

# 我可以免费获得 Live2D 模型吗？

如果你想成为 (Live2D) VTuber，最好有一个你认同的模型，通常是你创建的原创角色。使用互联网上的免费示例模型可能无法为你提供所需的体验。

但如果你只是想尝试一下 VTuber 或者正在寻找能在 VTube Studio 中进行测试的模型，你可以看一看官方 Live2D 示例模型: https://www.live2d.com/zh-CHS/learn/sample/

或者，如果你想成为 3D VTuber，你可以看看 VRoid 等应用程序，它可以让你轻松创建可在 VSeeFace 等应用程序中使用的 3D VTuber 模型: https://vroid.com/en
