你可以将 VTube Studio 配置从一个模型复制到另一个模型。如果你有多个具有相似Live2D参数的模型，这将非常有用。

你可以从 **“自动配置”** 菜单中访问此菜单。首先，加载要导入配置的模型。然后转到 **“自动配置”** 并选择 **“从其他模型复制配置”** 。这将显示 VTube Studio 中所有模型的列表，你可以选择 **一个模型进行复制** 。

![从其他模型复制配置截图](img/img/copy_setup_from_model_1.png)

## 可以被复制的内容

你可以在模型之间复制以下内容:

* **[基础设置]**

  这将复制基础模型设置，例如“移动配置”和空闲动画名称。

* **[物理演算/风模拟设置]**

  这将复制与物理和风模拟相关的所有模型设置。

* **[参数设置]**
 
  这会复制所有参数设置，以便将哪些跟踪（输入）参数映射到哪些 Live2D（输出）参数。

* **[快捷键]**

  这将复制与快捷键相关的所有设置。

* **[按键表情文件]**
  
  这将从模型中复制所有按键表情文件 (` .exp3.json`)。

* **[动画文件]**
  
  这将从模型中复制所有动画文件 (` .motion3.json`)。

* **[模型图标文件]**
  
  这将复制模型图标文件。

* **[ArtMesh设置]**
  
  这会复制所有 ArtMesh 数据，例如挂件固定排除、光系统自定义等。

如果目标模型（复制到的模型）的按键表情和动画文件位于子文件夹中，VTube Studio 也会自动将所有新动画文件复制到该文件夹​​中。

## 如果模型具有不同的 Live2D 参数怎么办？

仅当源模型（被复制的模型）和目标模型（复制到的模型）具有相同的 Live2D 参数时，模型设置、动画和按键表情才能正常工作。否则，模型设置中将缺少这些参数（它们将在目标模型中标记为红色），并且动画/表达式将忽略这些参数。

如果你激活 **[重命名丢失的参数]** ，VTube Studio 将通过检查不同的参数大小写和参数ID样式来自动尝试查找缺失的Live2D参数。例如，如果源模型使用旧的参数 ID 样式（如 PARAM_ANGLE_X ），而目标模型使用新的参数 ID 样式（如 ParamAngleX ），则参数在复制后将在模型设置、表达式和动画中自动更正。

关于Live2D默认参数ID的更多信息，请查看以下页面: https://docs.live2d.com/zh-CHS/cubism-editor-manual/standard-parameter-list/
