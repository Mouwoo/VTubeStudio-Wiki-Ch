    常见问题（FAQ）

    直播时我的面部会意外暴露吗？¶

    不会，这种情况绝不可能发生。 VTube Studio 仅将智能手机摄像头用于面部追踪，不会存储面部信息，也不会向电脑传输原始面部追踪数据，仅会传输计算后的参数。因此，电脑端始终不会出现任何面部视频画面。电脑/苹果电脑版本也完全没有调取摄像头画面并显示的功能。

    若你希望智能手机屏幕上也绝不显示面部画面，请将「摄像头预览」设置为「永不显示」（详见 「摄像头设置」）。

    别太担心别人会看到你的脸啦。你超好看的！♥

    如何购买 VTube Studio？为什么要购买？可以商用吗？¶

    若你将 VTube Studio 用于商业用途（如 YouTube 超级留言、Twitch 盈利直播等），则必须购买至少一个付费版本。具体购买哪个版本可自行决定，此为 Live2D 授权协议的强制要求。

    购买 VTube Studio 有两种方式，具体取决于你的使用需求：

    • 我仅需使用苹果手机/安卓手机追踪功能
    • 首先，从苹果应用商店或安卓应用商店免费下载 VTube Studio 应用，进入应用设置界面，点击显眼的「购买专业版」按钮即可完成购买。
    • 专业版将解锁免费版不具备的功能，例如：向电脑推流时长可超过 5 分钟、移除应用内的 VTube Studio 水印。

    • 此时你无需购买 Steam 版本，直接免费下载使用即可。使用苹果手机/安卓手机追踪功能时，不会显示任何水印。

    • 我仅需使用摄像头追踪功能

    • 从 Steam 免费下载应用，即可试用摄像头追踪功能。需要注意的是，仅在摄像头追踪功能激活时，画面会显示水印。
    • 购买「移除水印」DLC 即可去除该水印。

    若你同时需要使用摄像头追踪和苹果手机/安卓手机追踪功能，则需同时购买手机端和 Steam 端的 VTube Studio。

    我知道这个水印超可爱，但还是请你支持正版购买 VTube Studio！应广大用户需求，现在你可以在设置中取消勾选「隐藏 VTS 水印」，让水印角色转过身去。你也可以通过挂件系统，将她作为挂件添加到场景中哦。 顺带一提，这个水印的动画是由 Walfie（@walfieee）制作的。

    以上条款仅适用于个人创作者、小型虚拟主播经纪公司及一般小型企业。大型企业如需使用，则需额外获取授权。更多详情请参阅 VTube Studio 用户许可协议，或直接联系我们。

    • 用户许可协议（EULA）：https://denchisoft.com/license/

    • 联系方式：denchi@denchisoft.com

    购买 VTube Studio 前可以试用吗？¶

    当然可以，而且非常建议你先试用！

    安卓和苹果手机端的应用本体免费，且设有专业版升级选项。免费版包含所有功能（包括加载自定义模型），建议你先试用，再决定是否购买。

    Steam 端的应用本体同样免费，且包含所有功能。仅当你需要去除摄像头追踪时的水印时，才需要购买对应的 Steam DLC。

    可以不用智能手机，改用摄像头进行面部追踪吗？¶

    可以，Steam 版本支持摄像头追踪功能。

    如何与朋友一起使用 VTube Studio？¶

    你可以通过运行 start_without_steam.bat 批处理文件，多次启动 VTube Studio。之后，你可以使用多个摄像头或多部智能手机来分别控制不同的模型。你也可以用单个摄像头，同时控制多个 VTube Studio 实例。

    若你想和朋友通过互联网进行协作直播，可以借助 Zoom、Discord、Skype 等软件的群组视频通话功能实现。

    以 Discord 为例，所有人只需开启 VTube Studio 的虚拟摄像头功能，然后在 Discord 中将该虚拟摄像头选为视频通话的主摄像头即可。你也可以先用 OBS 进行录制，再通过 OBS 虚拟摄像头插件生成摄像头流，供 Discord 使用。协作直播的「主持人」可以用 OBS 录制 Discord 窗口，裁剪出每个人的画面，并使用色度键滤镜去除背景。这种方法效果很不错，但为了保证画质，你可能需要升级 Discord 的 1080p 高清视频通话功能。

    注意（实验性功能）：你也可以借助「零层一号」（Zero Tier One）或「蛤蟆吃」（Hamachi）等局域网穿透工具，通过互联网使用 VTube Studio（免费版最多支持 5 人）。你可以在「主机」电脑上启动多个 VTube Studio 实例，然后让其他人连接这些实例。切记要为每个实例设置不同的端口！

    免责声明：我本人并未测试过这种方法，但有用户反馈该方法可行。当然，这种方式可能会存在网络延迟。

    苹果手机应用能启动，但摄像头预览仅显示灰色画面，或仅显示摄像头画面但面部没有追踪蒙版，该怎么办？¶

    首先，关闭并重启 VTube Studio，然后重启你的苹果手机。若问题仍未解决，请彻底删除手机中的 VTube Studio 应用并重新安装，通常这样就能解决问题。如果还是不行，请在 VTube Studio 的 Discord 服务器中发帖求助。

    可以不用无线网络，改用 USB 线连接智能手机吗？¶

    若你使用苹果手机/平板进行面部追踪，则可以通过 USB 线连接。安卓设备暂不支持通过 USB 线直接传输面部追踪数据。USB 连接稳定性极高，是传输面部追踪数据的推荐方式。

    对于安卓设备，你可以尝试一种替代方案：通过 USB 线连接手机和电脑，然后开启 USB 网络共享功能，这样即使没有无线网络，两者也能处于同一网络环境中。

    重要提示：此内容与 VTube Studio 本身无关，但当你将苹果手机通过 USB 线连接到电脑/苹果电脑，且手机的 个人热点 处于开启状态时，电脑会自动尝试通过苹果手机的移动数据连接互联网。相关说明可参考：https://apple.stackexchange.com/questions/224069/iphone-prevent-automatic-hotspot-tethering-when-connecting-to-computer

    当我尝试通过 USB 线连接苹果手机时，Windows 电脑会死机，或帧率骤降至 1-2 帧，该怎么办？¶

    重启 VTube Studio 和电脑，然后尝试启动 iTunes。若问题仍未解决，请关闭 iTunes 和 VTube Studio。

    接着，打开文件资源管理器，在地址栏输入 %ProgramData%\Apple\Lockdown，删除该文件夹内的所有内容，之后再次尝试启动 iTunes 和 VTube Studio。

    什么是「虚拟摄像头」功能？¶

    「虚拟摄像头」（仅支持 Windows 系统）功能可以录制 VTube Studio 的窗口画面（不包含界面元素），并直接将其作为摄像头流输出，方便你在 Discord、Zoom 等软件中使用。

    使用前，你需要点击 「安装」 按钮来安装虚拟摄像头驱动。点击后会弹出一个 Windows 控制台窗口，系统可能会请求管理员权限，请务必授予。安装成功后，勾选「激活虚拟摄像头」选项，即可启动摄像头流。

    之后，你就能在 Discord、OBS、Zoom 等软件中，像选择普通摄像头一样选择该虚拟摄像头。该虚拟摄像头的名称为 「VTubeStudioCam」。虚拟摄像头同样支持透明背景推流。需要注意的是，虚拟摄像头的视频画质，可能略低于直接用 OBS 录制 VTube Studio 窗口的画质。

    若安装失败，请确保以管理员身份启动 VTube Studio。若虚拟摄像头仍未显示（例如在 OBS 中），请确保 OBS 也以管理员身份启动。

    我的设备会变得很烫！手机应用中的「推流模式」是什么？¶

    这种情况确实难以避免。你的设备可能会变得非常烫，但只要设置正确，就不会出现过热、崩溃等严重问题。这种情况在较旧的支持设备上尤为明显，例如 iPhone X。

    在向电脑推流时，你可以通过以下三种方法来降低设备发热：

    • 在智能手机上卸载 Live2D 模型：向电脑推流时，手机端无需加载模型。在手机上保持模型加载状态会消耗大量性能，对部分智能手机来说负担过重。

    • 关闭摄像头预览：长时间在应用中渲染摄像头预览画面，也会引发一些性能问题，建议将其关闭。

    • 将屏幕亮度调至最低：这能显著降低手机的发热量。

    采用以上设置后，连续推流 15 小时以上完全不成问题（记得保持手机充电状态）。该结论经 iPhone X 和 iPad 设备测试验证。

    「推流模式」 会在手机连接电脑时，自动卸载模型、关闭摄像头预览并调低屏幕亮度。建议你在向电脑推流时，始终开启推流模式。

    苹果电脑版本和 Windows 版本有区别吗？¶

    Windows 版本支持所有功能。苹果电脑版本不支持在 OBS 中实现透明背景，且不支持后台热键功能。这意味着，在苹果电脑上，只有当 VTube Studio 窗口处于激活状态时，热键才能生效。例如，当你同时运行游戏时，热键将无法触发。

    此外，苹果电脑版本的摄像头追踪功能尚处于高度实验性阶段，且很可能会一直保持该状态。使用苹果电脑版本的 VTube Studio 时，请务必留意这一点。

    在 Windows 电脑上运行应用时，耳机/录音中出现音频爆音，该怎么办？¶

    目前暂不清楚具体原因，过去有部分用户遇到过此类问题。通常，更新显卡驱动即可解决该问题。

    为什么应用会卡顿？为什么会占用大量的 CPU/GPU 资源？¶

    卡顿问题几乎都是由网络问题引起的。首先，请尝试重启路由器/交换机、电脑和智能手机。若问题依旧存在，请查阅「连接问题与故障排除」章节。

    若你遇到 CPU/GPU 占用率过高的情况，这属于异常现象。VTube Studio 本身并不需要高性能的 CPU/GPU 支持，出现这种情况很可能是哪里出了问题。若你遇到性能问题，可在 VTube Studio Discord 服务器 中寻求帮助。

    有时，「雷蛇游戏加速」（Razer Game Boost）或「微星小飞机」（MSI Afterburner）等 游戏性能优化工具 也可能导致性能问题，建议你尝试关闭这些工具，看看能否改善情况。

    还有用户反馈，开启 NVIDIA Reflex 功能后，CPU/GPU 占用率过高的问题得到了解决，你也可以尝试这个方法。

    可以使用单反相机进行摄像头面部追踪吗？¶

    理论上是可行的，但有用户反馈存在一些问题。对于部分单反相机，你可能需要关闭 「USB 选择性挂起」 功能。若相机仍无法在 VTube Studio 中正常工作，可以尝试通过 OBS 进行中转：先用 OBS 捕获相机画面，再通过 OBS 虚拟摄像头功能将画面输出，最后让 VTube Studio 读取该虚拟摄像头的画面。这种方法可能会对部分电脑的性能造成影响，但通常可以解决问题。

    为什么应用的帧率会高得离谱（超过 400 帧）？¶

    这是一个由显卡驱动程序导致的漏洞：部分显卡驱动会错误地报告显示器刷新率，导致 VTube Studio 开启垂直同步（VSync）后，帧率限制被解除，应用会试图以硬件支持的最高速度进行渲染。

    要解决这个问题，可以在英伟达显卡设置中开启 G-SYNC 功能（其他品牌显卡可开启对应的等效功能）。若问题仍未解决，可以在 VTube Studio 的设置中，将帧率模式从「垂直同步」改为固定 60 帧。

    我在安卓设备上购买了专业版，可以转移到苹果手机或 Steam 平台吗？¶

    一般情况下，不可以。从技术层面来说，由于谷歌和苹果使用不同的应用商店，转移授权是无法实现的。 不过，我可以提供一个为期一个月的宽限期。

    若你在安卓设备上购买了 VTube Studio，希望将授权「转移」到苹果手机上，可以通过邮件或 Discord 联系我。联系时，请先在苹果应用商店购买该应用，并提供购买凭证，我会为你办理安卓端的退款。 根据 Valve 的 DLC 相关政策，授权也不能在 Steam 平台之间或与其他平台互相转移。但如果你使用的是苹果手机/安卓手机追踪功能，则无需购买 Steam DLC——因为该 DLC 的唯一作用就是移除摄像头追踪时的水印，而使用手机追踪功能时，Steam 版本不会显示任何水印。

    VTube Studio 在我的智能手机上无法运行，该怎么办？¶

    请参阅「简介与系统要求」章节。

    苹果手机/平板的面部追踪功能完全失灵，这是怎么回事？¶

    首先，尝试重启手机。

    若重启后问题仍未解决，且摄像头预览中的追踪蒙版出现乱飞、随机卡顿等现象，通常是由电磁干扰引起的。苹果手机/平板内置磁力计（电子罗盘），用于计算设备的朝向。请确保你的设备没有使用带有磁性部件的保护壳，且没有靠在电脑显示器上。将设备放置在无线充电器上，也可能引发类似问题。

    在某些情况下，光线条件不佳也会导致该问题。请确保你的追踪环境光线充足且光线颜色自然，某些颜色可能会干扰苹果/安卓系统的面部追踪功能。你可以尝试换一个房间，看看问题是否有所改善。

    还有用户反馈，若摄像头画面中存在镜子，也可能会干扰追踪功能。

    为什么我的苹果手机/平板会调用前置摄像头？¶

    这说明你的设备不受支持。应用启动时，你应该也会看到相关的提示弹窗。支持的设备列表，请参阅「简介与系统要求」章节。

    苹果电脑和 Linux 系统是否受支持？搭载 M1 芯片的苹果电脑呢？¶

    • Linux 系统：暂无官方支持，但你可以通过「网络追踪器」模式手动启动 OpenSeeFace 面部追踪程序，从而在 Linux 系统上运行 VTube Studio，具体方法可参考这个 Trello 任务：https://trello.com/c/kbZWaoKN/26-variable-ip-for-osf-tracker。若你有相关问题，可在 Discord 中咨询。你也可以在「在 Linux 系统上运行 VTS」页面中找到更多信息。

    • 苹果电脑系统：实验性支持，且存在功能限制。你可以通过 Steam 正常下载应用，但需要注意，部分功能可能无法使用，包括稳定的摄像头追踪和音频唇形同步功能。搭载 M1 芯片的苹果电脑理论上可以运行，但可能存在更多的稳定性问题。对于苹果电脑版本，我只能提供非常有限的技术支持。

    使用虚拟摄像头或 NDI 功能时，音频出现卡顿/失真，该怎么办？¶

    这种情况可能在特定硬件组合下发生，例如使用 GoXLR 声卡时。将电脑的 BIOS 更新至最新版本，通常可以解决该问题。

    企业可以使用 VTube Studio 吗？用于电视广告也可以吗？¶

    这取决于企业的规模，大型企业可能需要额外获取授权。更多详情请参阅 VTube Studio 用户许可协议，或 通过邮件联系我。

    无法在 VTube Studio 中加载 Live2D 模型，问题出在哪里？¶

    首先，请查看日志文件，确认是否存在错误信息。若日志中显示错误，但应用未弹出错误提示窗口，则可能是应用存在漏洞。请通过 Discord 联系我，我会及时处理。

    也有部分案例显示，杀毒软件会阻止应用读取模型文件。你可以尝试暂时关闭杀毒软件，或为 VTube Studio 添加信任例外。例如「IObit 恶意软件清理助手」（IObit Malware Fighter）就曾出现过此类情况。

    若你在苹果手机/安卓手机应用中加载模型时，应用立即崩溃，这很可能是因为模型的纹理贴图尺寸过大。16K 分辨率的纹理贴图，或三张及以上的 4K 分辨率纹理贴图，对大多数移动设备来说负担过重。使用大尺寸纹理贴图，并不能提升模型的显示效果，因此请在 Live2D 中导出模型时，减小纹理贴图的尺寸。一般来说，一张或两张 4K 分辨率的纹理贴图完全足够使用。

    可以同时使用苹果手机/安卓手机追踪和摄像头追踪功能吗？¶

    虽然不太清楚你为什么需要这样做，但答案是可以的。若你同时激活了两种追踪功能，只要摄像头能够捕捉到你的面部，VTube Studio 就会优先使用摄像头追踪数据。一旦摄像头失去追踪，VTube Studio 会自动切换为使用网络传输的手机追踪数据。

    可以在 Linux 系统上结合 OpenSeeFace 追踪功能使用 VTube Studio 吗？¶

    Linux 系统暂无官方支持，但已有部分用户成功实现了该功能。

    更多相关信息，请查阅「在 Linux 系统上运行 VTS」页面。

    我使用的是苹果电脑系统，摄像头追踪功能无法使用。¶

    苹果电脑版本的摄像头追踪功能尚处于实验性阶段，可能会因设备型号不同而完全无法使用。购买任何 DLC 之前，请务必确认你对苹果电脑版本的追踪效果满意，因为我无法保证该功能能够稳定运行。最坏的情况是，若 VTube Studio 或其他应用无法再访问摄像头，你可能需要按照以下方法重置系统管理控制器（SMC）：

    系统管理控制器（SMC）重置指南：https://support.apple.com/en-gb/HT201295

    话虽如此，还是有不少用户在苹果电脑上顺利使用 VTube Studio，且没有遇到任何问题。苹果电脑版本搭配苹果手机/安卓手机追踪功能，应该可以正常使用。

    我的模型上出现奇怪的黑线，尤其是嘴巴运动时，黑线会更加明显，这是什么原因导致的？¶

    这很可能是因为你在显卡设置中关闭了 各向异性过滤 功能。该功能必须处于开启状态，模型才能正常渲染。简单来说，这个设置用于控制纹理贴图在大角度形变时的渲染效果。

    请确保各向异性过滤设置为「由应用程序控制」「8 倍」或「16 倍」。

    不同显卡驱动的设置方法示例，请参考：https://www.pcgamingwiki.com/wiki/Glossary:Anisotropic_filtering_(AF)

    对于英伟达显卡，你可以在「英伟达控制面板」中进行设置，请确保该选项 没有 设置为「关闭」。

    VTube Studio 运行一段时间后，会随机崩溃，该怎么办？¶

    这种情况通常是由硬件问题引起的，常见于内存、CPU 或主板故障。

    你可以查看 Windows 事件查看器，若崩溃原因显示为 error 0xc0000005，则可确认是硬件问题。

    在某些情况下，修改 BIOS 设置可以解决该问题。你可以尝试的完整解决方案列表，请查阅 此页面。

    VTube Studio 会存储/记录个人数据吗？¶

    不会。

    具体来说，VTube Studio 不会在设备上存储任何个人数据，也不会向任何外部服务器发送个人数据。VTube Studio 会在电脑上生成日志文件，但这些日志均为人类可读的文本文件，且不会自动发送给任何人，除非你手动上传。你也可以随时删除这些日志文件。

    更多信息，请参阅官网的隐私政策：https://denchisoft.com/privacy/
